# Project: AI API Pricing Calculator

## 現在の状態

2026-05-02 時点で、静的MVPの初期構成を作成済み。

## 目的

AI APIやAI SaaSの料金を比較し、月間利用量から概算費用を計算できるサイトを運用する。

## 公開方針

- 第一候補: Cloudflare Pages
- 初期は静的サイトのみ
- 動的サーバ、DB、ログイン機能は持たない
- 価格データは `data/pricing.json` で管理

## 初期MVPの機能

- 月間リクエスト数から概算費用を計算
- 入力/出力トークン単価を比較
- キャッシュ入力を計算に反映
- Batch割引を計算に反映
- USD/JPY換算
- 公式価格ページへのリンク
